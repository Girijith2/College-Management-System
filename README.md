# College-Management-System

This project is a simple **College Management System** developed using  
**Java** to demonstrate basic concepts of object-oriented programming,  
class structure, and data handling in a console-based application.

The system manages essential college information such as student details,  
course information, and basic operations through a structured Java program.  
It is designed as a beginner-friendly academic project to practice core  
Java programming concepts and logical implementation.
